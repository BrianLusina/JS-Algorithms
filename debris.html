Task:

Map The Debris

Return a new array that transforms the element's average altitude into their orbital periods. The array will contain objects in the format {name: 'name', avgAlt: avgAlt}. You can read about orbital periods on wikipedia The values should be rounded to the nearest whole number. The body being orbited is Earth. The radius of the earth is 6367.4447 kilometers, and the GM value of earth is 398600.4418 km3s-2.

function orbitalPeriod(arr) { var GM = 398600.4418; var earthRadius = 6367.4447; var result = []; var pi2 = 2 * Math.PI;

for(var x = 0; x< arr.length; x++){ var eRadAlt = Math.pow(earthRadius + arr[x].avgAlt,3); var c = Math.sqrt(eRadAlt/GM); arr[x].orbitalPeriod = Math.round(pi2 * c); delete arr[x].avgAlt; result.push(arr[x]); } return result; }
